String-format

A much better alternative than String.format() for big strings.

View the Project on GitHub JoanZapata/string-format

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Usage

Strings.format("Hello {firstname} {lastname}!")
    .with("firstname", "John")
    .with("lastname", "Doe")
    .build();

Get it

Download JAR or via Maven Central

<dependency>
    <groupId>com.joanzapata.utils</groupId>
    <artifactId>string-format</artifactId>
    <version>1.0.0</version>
</dependency>
